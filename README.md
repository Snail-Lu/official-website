# OFFICIAL WEBSITE

基于 `Vue3` + `Vite` 搭建的纯静态官网项目。[线上地址](https://snail-lu.github.io/official-website/#/)

## 项目实现

-   移动端适配 ---- CSS Media Queries
-   路由 ---- vue-router
-   全局状态管理 ---- vuex
-   代码格式化 ---- prettier
-   图标 ---- iconfont
-   国际化 ---- vue-i18n

## 基本使用

```bash
# 克隆代码
git clone https://github.com/Snail-Lu/official-website.git

# 安装依赖
cd official
npm install

# 本地运行
npm run dev

# 生产打包
npm run build

# 部署到 github pages
npm run deploy
```
